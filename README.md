# NFT Project

A simple ERC-721 NFT smart contract built with Solidity, Hardhat, and OpenZeppelin.

## Features

- ERC-721 compliant
- Metadata support
- Owner-only minting
- Easy deployment with Hardhat

## Installation

```bash
npm install
```

Compile:

```bash
npx hardhat compile
```

Run tests:

```bash
npx hardhat test
```

Deploy:

```bash
npx hardhat run scripts/deploy.js
```
